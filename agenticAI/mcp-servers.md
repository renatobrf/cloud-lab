# MCP Server
Um MCP Server (Servidor de Protocolo de Contexto de Modelo) é um programa que conecta modelos de IA (LLMs, como Claude ou ChatGPT) a dados e ferramentas externas de forma padronizada. Ele funciona como um intermediário que permite à IA acessar bancos de dados, arquivos locais, APIs (como Google Calendar) ou CRMs para obter informações em tempo real e executar tarefas.

## Padroes
O Model Context Protocol (MCP) é um padrão aberto lançado pela Anthropic que permite conectar IAs (LLMs) a fontes de dados externas e ferramentas de forma segura e padronizada. Ele atua como um "conector universal", permitindo que modelos acessem bancos de dados, arquivos e APIs, melhorando a relevância das respostas e a automação de tarefas.
